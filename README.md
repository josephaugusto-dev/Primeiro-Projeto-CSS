## 📋 Sobre o Projeto

Este foi meu **primeiro projeto** desenvolvido com HTML5 e CSS3. Representa o início da minha jornada no desenvolvimento web, onde comecei a entender os fundamentos de estilização, seletores, propriedades e o modelo de caixas do CSS.

---

## ✨ O que foi aprendido

- Seletores CSS (classe, id, elemento)
- Modelo de caixa — `margin`, `padding`, `border`
- Cores, fontes e tipografia
- Estrutura básica de uma página HTML
- Primeiros passos com posicionamento
